Zombie horror survival

Build Date: 13.04.24 Category: 3D FPS Game Engine: Unity Language: C# Level Count: 3

Zombie survival is an ongoing project. I decided to make a survival shooter game as my first project. I have chosen a tutorial and going to follow it on my way to building a complete game. The game is just a common horror shooter where you have to escape from evil creatures while collecting necessary items on your way. I would like to create a simple multiplayer mode because I have never done this before. I am mostly inspired by the "Resident Evil" and "Left4Dead".

The game was created using Unity Engine and C# programming language. 3d models, animations and sounds are taken from free sources and my part is to code only.

GitHub repository

Project Overview

The project is made only by me as a solo developer with a help of tutorials and free assets.

The project is not yet available for download, but a demo is available upon request.

I implemented advanced AI pathfinding algorithms that adapt to dynamic in-game changes, creating a fluid and engaging player experience.

One significant challenge was ensuring smooth gameplay on lower-end devices. This was overcome by optimizing asset usage and employing efficient coding practices. Animations were also a challenge, but I was able to overcome this by using Unity's built-in animation tools.

Comprehensive testing was conducted, with documentation available upon request.

I am proud of the progress I have made on this project and am excited to continue developing it further. I have learned a great deal about game development and how Unity works and am eager to apply this knowledge to future projects.
