Uncircling

Build Date: 07.10.23 Category: 2D Game Engine: Unity Language: C# Difficulty level: 1

Uncircling is a 2d Unity game where you have to avoid obstacles and enemies.

The game was created for a 2 day gamejam using Unity Engine and C# programming language. It has a simple and intuitive control system, and a mechanic when objects around player can be picked up and used as a weapon.

It was made as a group of three people.

GitHub repository Itch.io

My Contribution

I was responsible for player movement script, enemy AI, enemy shooting and random loot appearence from the boxes. With second programmer from our team we created the mechanic of spinning objects around our character. Music and sound effects were also made by me, but it had not been added to the final game because of the lack of time.

One significant challenge was to make the revolution mechanic together, because it required math knowledge and how to implement it into the code. As my first gamejam project, I was thrilled to learn how to work as a team and at the same time plan my own work

This project significantly honed my problem-solving skills and my ability to write clean, maintainable code. It was a true testament to the power of teamwork and perseverance. GitHub also helped me to understand how to work with version control and how to work with other people's code and share my own code.
